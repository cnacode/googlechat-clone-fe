## Project Structure
    |-- app
        |-- Components
            UI components ready to be used in pages
        |-- Core
            Services, Utilities, Interceptors etc. Go here
        |-- Page
            Application pages go here. Highest level of logic
        |-- Shared
            tiny bits of ui logic, designed to be reusable in Components
